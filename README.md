# Linux SMTP Server

Socket-based SMTP server implementation developed in C for Linux environments. The project demonstrates basic email server operations, socket programming, SMTP communication, and Linux network programming concepts.

## Features

- SMTP server implementation
- Socket-based client/server communication
- Email receiving operations
- Command-line port configuration
- Linux network programming
- File-based email storage
- Makefile-based build automation

## Technologies

- C Programming
- Linux
- Socket Programming
- SMTP Protocol
- TCP/IP Networking
- Makefile
